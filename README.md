# BrainTeaserPuzzleGame
Java Swing puzzle game with rotation, drag-and-drop, and multiple difficulty levels.
## Features

- Interactive **drag-and-drop** for puzzle pieces
- **Right-click rotation** of pieces
- Dynamic difficulty levels: **Easy (3x3), Medium (6x6), Hard (9x9)**
- **Custom image selection** for puzzles
- **Move counter** and **timer** with completion feedback
- Ensures all puzzles are **solvable** using inversion-based logic
